# Responsive Product Webpage

This project is a responsive single-page website built using **Vanilla HTML, CSS, and JavaScript** as part of a frontend assignment.

The page is implemented based on the provided Figma design and includes interactive UI elements such as a sticky header, image carousel, and FAQ section.

## Live Demo
https://kavya-n03.github.io/frontend-webpage/

## GitHub Repository
https://github.com/Kavya-N03/frontend-webpage

## Features
- Responsive layout (Desktop, Tablet, Mobile)
- Sticky header on scroll
- Image carousel
- Zoom effect on product image
- FAQ accordion interaction
- Testimonials slider
- Clean and structured layout

## Technologies Used
- HTML5
- CSS3 (Flexbox / Grid)
- JavaScript (Vanilla JS)
